============================================================
                 Maxwell–Dice System (Paper Form)
============================================================

Author: Maximillian
Purpose: A physics-inspired dice system using the discrete
         Maxwell–Boltzmann distribution to generate rolls
         for d9 and d12 using true randomness.

============================================================
1. Introduction
============================================================

Maxwell–Dice is a stochastic dice-rolling system based on the
Maxwell–Boltzmann speed distribution. Instead of assigning
equal probability to each face, the system uses a physically
motivated probability curve that favors mid-range outcomes.

The system is defined for dice with N faces, typically N = 9
or N = 12.

============================================================
2. Maxwell–Boltzmann Distribution (Discrete Form)
============================================================

For a die with faces i = 1, 2, ..., N, define the discrete
Maxwell–Boltzmann weight function:

    f(i) = sqrt(2/pi) * (i^2 / a^3) * exp(-i^2 / (2a^2))

where:

    i  = face index
    a  = scale parameter ("temperature")

These weights are not yet probabilities. Normalize them:

    w_i = f(i)

    p_i = w_i / (sum over j=1..N of w_j)

This produces a probability distribution over the faces.

============================================================
3. Cumulative Distribution Table
============================================================

Define the cumulative distribution:

    P_k = sum(p_i for i = 1..k)

This table is used to map a random number x ∈ (0,1] to a
discrete face.

============================================================
4. Precomputed Tables for d9 and d12
============================================================

For convenience, the following cumulative tables are used:

d9 cumulative MB table:
    P9 = [0.040, 0.183, 0.423, 0.663, 0.835,
          0.931, 0.975, 0.992, 1.000]

d12 cumulative MB table:
    P12 = [0.020, 0.070, 0.160, 0.290, 0.450,
           0.620, 0.780, 0.890, 0.950,
           0.980, 0.995, 1.000]

============================================================
5. Randomness Model
============================================================

Maxwell–Dice uses true randomness:

    x ~ Uniform(0, 1]

This value is generated fresh for each roll.

============================================================
6. Roll Procedure
============================================================

Given a cumulative table P_k and a random value x:

    Find the smallest k such that P_k >= x

Output k as the Maxwell–Boltzmann roll.

============================================================
7. Example
============================================================

If x = 0.37 and using d9:

    0.183 < x <= 0.423  →  face 3

============================================================
8. Summary
============================================================

Maxwell–Dice replaces uniform dice with a physics-based
distribution. Rolls are generated by sampling a random value
and mapping it through the cumulative Maxwell–Boltzmann curve.

This system produces more mid-range results and fewer extremes,
matching the statistical behavior of particle speeds in a gas.
